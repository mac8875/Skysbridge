# Sky's Bridge Foundation website

This is a deploy-ready static prototype for skysbridge.org.

## Files
- index.html — main website
- styles.css — responsive design
- app.js — prototype interactions
- privacy.html — draft privacy principles
- community-guidelines.html — draft community rules
- assets/skysbridge-logo.png — selected original logo

## Publish on a hosting provider
Upload the entire folder without changing its structure.

Common options:
1. Netlify: drag the folder into Netlify Drop.
2. Vercel: import the folder as a static project.
3. Your domain registrar: connect skysbridge.org using the DNS records supplied by the host.

## Important before public launch
This prototype does not store accounts, memorials, messages or images.
Before adding real users, obtain professional review for:
- UAE and international privacy compliance
- safeguarding and moderation
- crisis escalation
- terms of use and consent
- secure authentication and database design
- public fundraising permissions

The names Mila and Little Lion are fictional placeholders.
