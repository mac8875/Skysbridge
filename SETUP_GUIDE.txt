NO MANUAL SQL EDITOR IS REQUIRED FOR THE INITIAL MIGRATION

1. Extract this ZIP file.
2. Upload ALL files and folders to the ROOT of the GitHub repository.
3. Important: preserve the folder path:
   supabase/migrations/20260724180000_initial_platform.sql
4. Commit the files.
5. Merge them into the production branch configured in Supabase.
6. Supabase will apply the migration through the GitHub integration.
7. Netlify will deploy the website from the same repository.
